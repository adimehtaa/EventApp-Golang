# Event Management REST API (Go + Gin)

A simple REST API for managing events, users, and attendees built with Go and the Gin framework.  
This project demonstrates database migrations, authentication with JWT, authorization, and API documentation using Swagger.

---

## Features

- User registration and login
- JWT-based authentication
- Authorization for protected routes
- CRUD operations for events
- Connect attendees to events
- SQL database with migrations
- RESTful API design
- Swagger API documentation
- API testing with Curl

---

## Tech Stack

- **Go**
- **Gin** (Web Framework)
- **SQL** (SQLite)
- **JWT** (Authentication)
- **Swagger / OpenAPI**
- **Curl** (API testing)
